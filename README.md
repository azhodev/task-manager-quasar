## Getting Started

### Install dependencies

```bash
npm install

```

### Start development server

```bash
npm run dev
```

### Build for production

```bash
npm run build

```

### Lint & Format code

```bash
npm run lint      # Lint with ESLint
npm run format    # Format with Prettier
```
